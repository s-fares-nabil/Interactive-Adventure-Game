# Interactive Adventure Game

A Python-based text adventure game with a custom GUI and 12 unique branching storylines. Features robust state management with 45 state variables across 8 distinct endings.

## Features

- Custom GUI interface built with Python
- 12 branching storylines
- 8 unique game endings
- 45+ state variables tracking player decisions
- Real-time state management

## How to Run

```bash
python main.py
```

## Requirements

- Python 3.x
- tkinter (included with standard Python installation)
